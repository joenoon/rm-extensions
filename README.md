# RMExtensions

Extensions and helpers for dealing with various areas of rubymotion:

- memory management
- block/scope/local variable issues
- GCD blocks
- retaining objects through async procedures

## Installation

Add this line to your application's Gemfile:

    gem 'rm-extensions'

And then execute:

    $ bundle
